# Bella Hair Studio 💇‍♀️

Este é o site institucional do **Bella Hair Studio**, um salão especializado em Mega Hair, localizado na Savassi, em Belo Horizonte. O projeto consiste em uma landing page responsiva, moderna e otimizada para conversão, facilitando o agendamento de clientes via WhatsApp e telefone.

## 🚀 Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias:

- **HTML5**: Estruturação semântica do conteúdo.
- **CSS3**: Estilização personalizada (arquivo `style.css`).
- **Bootstrap 5.3**: Framework para layout responsivo e componentes (Carousel, Grid, Utilitários).
- **Bootstrap Icons**: Biblioteca de ícones para links de contato e redes sociais.
- **Google Fonts / Web Fonts**: Para uma tipografia elegante e moderna.

## ✨ Funcionalidades

- **Design Responsivo**: Adaptado para dispositivos móveis, tablets e desktops.
- **Call to Action (CTA)**: Botões de agendamento rápido via WhatsApp e ligação.
- **Galeria de Resultados**: Carrossel de imagens exibindo transformações (antes e depois).
- **Tabela de Preços**: Listagem detalhada de serviços de kits, manutenção e aplicação.
- **Informações de Contato**: Localização integrada, horários de funcionamento e links para redes sociais (Instagram, TikTok).
- **Animações de Revelação**: Seções que surgem conforme o scroll do usuário (via classe `reveal`).

## 📂 Estrutura do Projeto

```text
bella-hair-studio/
├── index.html        # Página principal
├── style.css         # Estilizações customizadas
├── favicon.svg       # Ícone da aba do navegador
├── logo.svg          # Logotipo do estúdio
└── assets/           # Imagens e recursos visuais (imagem-1, 2, 3, etc.)
```

## 🛠️ Como Executar

1. Faça o clone ou download deste repositório.
2. Certifique-se de que todas as imagens citadas no HTML estão na pasta raiz (ou ajuste os caminhos).
3. Abra o arquivo `index.html` em qualquer navegador moderno.
